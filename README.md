# PAMS
个人账目管理系统

## Introduction

PAMS:Personal Accounts Management System.

I developed this project aimed to count my daily cost and warn myself how much I have paid this day/month/year. I hope it could help me learn to save my money.

If you have the same trouble with me, welcome to join me developing this project or use it without any pay. 

This project is written in PHP7.3. I try to use simple inside-function so theoretically it could running at most PHP server with great compatibility.

I start to written this project at Mar 19, 2021.
